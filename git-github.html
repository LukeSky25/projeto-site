Git + GitHub

Git é um software de versionamento de codigo distribuido VCS que cuida das versões dos códigos no seu repositório local e os envia para os servidores remotos. GitHub é como se fosse uma rede social de programadores onde você pode armanezar os seus códigos remotamente graças ao Git.
